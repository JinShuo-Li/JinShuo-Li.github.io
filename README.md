# My GitHub Pages

This repository hosts my personal website at [https://JinShuo-Li.github.io](https://JinShuo-Li.github.io). It's built using simple HTML and CSS, showcasing my about section, projects, and contact information.

## About the Site
- **Purpose**: A portfolio site to introduce myself, highlight my interests in Mathematics, Physics, Biology, and Computer Science, and display my GitHub projects.
- **Tech Stack**: Pure HTML5, CSS3, with Google Fonts for styling. Responsive design for mobile and desktop.
- **Built With**: GitHub Pages for free hosting.
